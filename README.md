# Input Source Handler

A simple macOS utility to switch input sources using custom keyboard shortcuts.

- Maps `Control + Shift + J` to the `かな` (Kana) key.
- Maps `Control + Shift + ;` to the `英数` (Eisu) key.

## Installation

```bash
# Enable my Homebrew Tap (First time only)
brew tap yuyuyuyuyu-dev/tap

# Install the app
brew install --cask input-source-handler
```

## Permissions

This application requires **Accessibility** permissions to intercept keyboard events. 

Please grant access to the app in:

`System Settings` > `Privacy & Security` > `Accessibility`
